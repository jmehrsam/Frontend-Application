Purpose

This website was designed with HTML, CSS and JS for the DI Frontend Application
Its goal is to create a website that takes userinput, call the number API and then display the returned text
The userinput is comprised of following data:
  • type
  • number

How-to

  1. Select a type from the dropdown (trivia is default)
  2. Type a number
  3. Click submit
  4. Read the output
