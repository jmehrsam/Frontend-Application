DOM = Document Object Model

The DOM is a model that represents all elements of an HTML document as objects.
In Javascript there are methods and events available to access and modify this object structure.
Through the DOM it is possible to:

  • add, modify and delete HTML elements
  • add, modify and delete attributes of HTML elements
  • add, modify and delete CSS statements
  • respond to HTML events
  • register, modify and delete HTML events
  • register, modify and delete HTML events
  • register, modify and delete HTML events
When an HTML document is loaded, the browser creates this object model.
Once this object model is created, it is possible to access it.
Changes to this object model have an immediate effect on the rendered HTML document, i.e. on the view in the browser.
