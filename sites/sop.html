SOP = Same-origin policy

Same-origin policy is a security mechanism that restricts how resources, such as a document or script, from one origin
can interact with a resource from another origin. It is considered to be the same origin if:

  • the protocol
  • port (if specified)
  • host
are the same for both.

SOP helps mitigating potetially documents and reducing possible attack vectors.
